## About This Project
I built this project to explore IPL data and find 
interesting patterns that aren't obvious just by 
watching matches. Being a cricket fan, I was curious 
whether toss actually matters, which players dominate 
consistently, and which teams have been truly successful 
over the years.

## What I Learned
- How to clean real-world messy data
- How to find insights using Pandas groupby and aggregations
- How to present data visually using Matplotlib and Seaborn
- Discovered that toss winning has almost no impact on match result!

## Challenges I Faced
- Same teams had different names in dataset
  (e.g. Delhi Daredevils vs Delhi Capitals)
- Fixed using Pandas replace() function
